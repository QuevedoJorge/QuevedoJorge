## 📌 Proyectos

🔹 **Sistema de reclutamiento**
Aplicación web para la gestión de candidatos y procesos de selección.
Incluye manejo de estados de postulaciones, seguimiento de candidatos y administración de procesos.
Stack: React, TypeScript, NestJS, Tailwind CSS

🔹 **EstacionaYa**
Plataforma enfocada en la renta de garages y espacios. Permite a los usuarios buscar, reservar y pagar alquileres por hora, día o mes, mientras que los arrendadores pueden publicar y gestionar sus espacios.

Incluye:

* Sistema de reservas con estados
* Cálculo automático de comisiones
* Feed personalizado de espacios disponibles

Stack: React, TypeScript, NestJS, Tailwind CSS
